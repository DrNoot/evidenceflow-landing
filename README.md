# EvidenceFlow Landing Page

React landing page for [EvidenceFlow](https://drnoot.github.io/evidenceflow-landing/), an AI-powered systematic review screening tool.

Built with Vite, React, TypeScript, Tailwind CSS v4, and select React Bits components (Aurora, BlurText, SpotlightCard, StarBorder, FadeIn).

## Development

```bash
npm install
npm run dev
```

## Deploy

Pushes to `main` trigger GitHub Actions to build and deploy to GitHub Pages automatically.
